No. コマンド 意味 備考
1 git branch ブランチ名 新しいブランチを作成する
2 git branch 作成したブランチを表示 git branch -a でリモートbranchも表示
3 git checkout ブランチ名 ブランチの変更 git checkout -b ブランチ名 で作成と変更を同時に行える
4 git merge ブランチ名 ブランチをoriginにマージする ワークツリーはoriginブランチで行う
5 git pull fetch, mergeを同時に行う 必ず master branch に pull されてしまうので注意が必要
